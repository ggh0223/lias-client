# 전체 결재 현황 페이지

## 개요

이 페이지는 모든 결재 상태의 문서를 한눈에 확인할 수 있는 대시보드입니다. 협의, 결재, 시행 데이터를 메인으로 보여주며, 각 상태별로 문서를 필터링하여 볼 수 있습니다.

## 주요 기능

### 1. 통계 카드

- 각 결재 상태별 문서 개수를 시각적으로 표시
- 협의 대기, 결재 대기, 시행 대기, 상신함, 기결함, 반려함, 수신참조함
- 카드 클릭 시 해당 상태의 문서 목록으로 이동

### 2. 탭 네비게이션

- 각 상태별 탭으로 문서 목록을 전환
- 탭에 문서 개수 표시
- 협의 대기, 결재 대기, 시행 대기가 상단에 배치되어 우선순위 표시

### 3. 문서 목록 테이블

- 문서 제목 및 문서 타입
- 작성자 및 부서 정보
- 작성일
- 현재 상태 배지
- 현재 단계 정보

## 상태별 우선순위

1. **협의 대기** - 협의가 필요한 문서
2. **결재 대기** - 결재 승인/반려가 필요한 문서
3. **시행 대기** - 결재 완료 후 시행이 필요한 문서
4. **상신함** - 내가 작성한 문서
5. **기결함** - 결재가 완료된 문서
6. **반려함** - 반려된 문서
7. **수신참조함** - 참조로 수신된 문서

## 기술 스택

- **React** - UI 컴포넌트
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 스타일링
- **Custom Hooks** - 데이터 관리 (`useAllApprovalDocuments`)

## API 연동

- `useAllApprovalDocuments` 훅을 사용하여 모든 상태의 문서를 동시에 조회
- 각 상태별로 병렬 API 호출
- 실시간 데이터 새로고침 기능

## 사용자 경험

- 로딩 상태 표시
- 에러 처리 및 메시지 표시
- 반응형 디자인 (모바일, 태블릿, 데스크톱)
- 호버 효과 및 인터랙션
- 직관적인 아이콘과 색상 코딩
